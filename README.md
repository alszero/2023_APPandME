### 앱앤미 웹사이트
미림여자정보과학고등학교의 전공동아리 앱앤미(AppandMe) 홍보 및 지원 사이트입니다.(2023.03 ~ 운영 중)

#### Website
http://appandme2023.dothome.co.kr/

### Features

  * 부원소개
    * 10기와 11기 부원들의 소개와 사진을 보여줍니다.
  * 지원하기
    * 앱앤미에 지원하려는 신입생들의 온라인 신청을 받습니다.
  * About
    * 앱앤미 동아리의 간단한 소개를 제공합니다.
  * 활동기록
    * 2018~2022년까지의 동아리 활동을 정리해두었습니다.
